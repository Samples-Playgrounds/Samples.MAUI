# CSnakesTest

Questo è un progetto di test per il framework CSnakes: https://tonybaloney.github.io/CSnakes/.
Consente di integrare direttamente in un progetto .Net un progetto Python, 
con il relativo ambiente, le librerie native ed il codice sorgente.
È necessario creare l'ambiente Conda "csnakes" con un comando simile:
```
conda create -n csnakes python=3.13.0 python pip
```
oppure
```
conda create --prefix PATH_TO_ENVIRONMENTS\csnakes python=3.13.0 python pip
```
